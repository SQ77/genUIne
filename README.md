## Getting Started
1. Clone the repo:  
   ```bash
   git clone git@github.com:SQ77/genUIne.git
   cd genUIne
   ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Start the development server:
    ```bash
    npm run dev
    ```
4. Scan the QR code using [Lynx Explorer](https://lynxjs.org/guide/start/quick-start.html) to run the app
